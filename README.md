# replog-site

筋トレ記録アプリ「**レップログ（RepLog）**」の公開サイト。

- 公開 URL: <https://replog.norudesign.com/>
- プライバシーポリシー: <https://replog.norudesign.com/privacy/>
- 利用規約: <https://replog.norudesign.com/terms/>
- 特定商取引法に基づく表記: <https://replog.norudesign.com/commerce/>

GitHub Pages（Jekyll）でホスティング。アプリ本体のコードは別リポジトリ。

## 構成

```
.
├── index.md            # トップページ
├── privacy/
│   └── index.md        # プライバシーポリシー
├── terms/
│   └── index.md        # 利用規約
├── commerce/
│   └── index.md        # 特定商取引法に基づく表記
├── _layouts/
│   └── default.html    # Cayman テーマオーバーライド
├── _config.yml         # Jekyll 設定
└── CNAME               # カスタムドメイン (replog.norudesign.com)
```
